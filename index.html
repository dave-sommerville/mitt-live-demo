Websites are built using HTML, or HyperText Markup Language, which defines the structure and content of a webpage. HTML uses elements such as headings, paragraphs, links, and images to describe what information appears on the page and how it is organized.

  • Content is defined using HTML elements
  • Structure creates meaning and hierarchy
  • Browsers interpret HTML first
  • CSS applies visual styling afterward
  • Design changes without touching content

CSS, or Cascading Style Sheets, controls how a webpage looks without changing its underlying content. It allows developers to apply colors, fonts, spacing, and layouts to HTML elements. CSS makes websites easier to update and more consistent across pages.
